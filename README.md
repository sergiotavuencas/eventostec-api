Este projeto foi desenvolvido com o objetivo de aprofundar os conhecimentos em Spring Boot e AWS, seguindo o tutorial 
ministrado por Fernanda Kipper. O tutorial visa desenvolver uma API para o cadastro de eventos tecnológicos, e realizar
o deploy da aplicação, servidor e banco de dados na AWS.

## Tecnologias

- Spring Boot
- Spring MVC
- PostgreSQL
- AWS (IAM, EC2, SC3, RDS e Secret Manager)
- Docker

## Práticas Adotadas

- API RESTful
- Computação em Nuvem
- Conteinerização
